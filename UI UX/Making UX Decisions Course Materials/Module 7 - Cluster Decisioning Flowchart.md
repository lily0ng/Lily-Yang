Develop a practical decisioning workflow for the Cluster project.

#### Exercise

Create a decisioning flowchart for making UI decisions on Cluster's content planning feature.

## Instructions

Review the Cluster PRD, focusing on the content planning feature.

Open the provided FigJam template.

Create a flowchart that represents your decision-making process for UI choices in Cluster's content planning feature.

Include decision points that incorporate institutional knowledge, customer familiarity, and user research.

## Completion criteria

Flowchart includes at least 5 decision points

Each decision point has clear "yes" and "no" paths

Flowchart incorporates all three levels of information (institutional knowledge, customer familiarity, user research)
